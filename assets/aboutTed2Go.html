Ted2Go is a fork of an official Ted2 IDE.

Sources on Github | Todo on Trello | Discuss on Forum

Features

  • Autocompletion List
    It based on real parsing of sources, parse all modules and user's code.
    (WIP)

  • CodeTree View
    Shows code structure, gives us comfortable code jumping.

  • Goto definition
    Press F2 to goto definition of ident under cursor.

  • Parsing on-the-fly
    You can see parse errors in realtime. Point the mouse to see a hint.

  • ToolBars
    MainToolBar - to simplify access to actions: open/save/undo/redo/build.
    EditorToolBar: find/comment/shift.

Third party

Almost all icons for CodeTree was taken from Netbeans IDE.
