Ted2Go

Ted2Go is a fork of an official Ted2 IDE.

Sources: https://github.com/engor/Ted2Go

Features

  • Autocompletion List
    It based on real parsing of sources, parse all modules and user's code.
    (WIP)

  • CodeTree View
    Shows code structure, gives us comfortable code jumping.

  • Extandable CodeEditor
    Editor consists of CodeFormatter, Keywords and Highlighter components, so you can write plugin for any language.
